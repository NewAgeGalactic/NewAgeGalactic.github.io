in order to learn laviera, there are rules that you must follow.

the first one that is common is

laviera's vowels

  • ay

these make up the basics of laviera, but there is still one left to learn.

double letters. double letters allow laviera to do translation from english to laviera and vice versa to be way easier

    laviera's double letters rules goes as follows
  • when 2 letters of the same type are next to each other, say "ll", instead of saying "lele", it will be read as "leh"
  • double letters can and will appear in words from english and laviera.

it's not a rare accurance that words will be shortened and because of that we have double letters to help!
